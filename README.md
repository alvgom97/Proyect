## Requirements

- Install [nodejs](https://nodejs.org/es/download) (You may need to restart afterwards)

- Install Angular
```
npm i -g @angular/cli
```
- [User Management Server](https://github.com/alvgom97/Users-Manager-Server)

## Initialize

- Install the project
```
npm i 
```
- Start project, click [here](https://localhost:4200)
```
ng serve 
```
